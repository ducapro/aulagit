Sobre a Turma

Sejam bem-vindos à nossa jornada no mundo do desenvolvimento web! Aqui, exploraremos os fundamentos do HTML, CSS e JavaScript para construir páginas e aplicações interativas.

Nosso objetivo é que todos saiam com uma compreensão sólida de como a web funciona e a capacidade de criar seus próprios projetos.
